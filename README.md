# 📖 Descripción

Este proyecto corresponde al **Taller 02** de la Unidad 3 de la asignatura **Técnica de Programación**.

El sistema fue desarrollado como una aplicación web que permite resolver un caso de estudio real mediante la utilización de:

- ✔️ Vectores (Arrays)
- ✔️ Ciclos (`for`)
- ✔️ Condicionales (`if`, `else`)
- ✔️ Funciones
- ✔️ Validación de datos
- ✔️ Manipulación del DOM

---

# 🎯 Objetivos

- Aplicar programación modular en JavaScript.
- Utilizar vectores para almacenar información.
- Validar los datos ingresados por el usuario.
- Automatizar cálculos mediante ciclos y condicionales.
- Mostrar resultados dinámicamente en la página web.

---

# 🚀 Funcionalidades

## Caso 1 - Simulador de Factura de Ventas

- Registro de productos.
- Ingreso de cantidades.
- Cálculo automático del subtotal.
- Cálculo del IVA (15%).
- Aplicación de descuento cuando corresponde.
- Cálculo del total a pagar.
- Resumen completo de la compra.

---

## Caso 2 - Cajero Bancario Inteligente

- Registro de depósitos.
- Registro de retiros.
- Validación de saldo disponible.
- Historial de movimientos.
- Total depositado.
- Total retirado.
- Saldo final.
